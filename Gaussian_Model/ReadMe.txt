README for Gaussian Programs
Forward_Functions contains the main model definition, training loop, prediction function, and standardisation
Read_Write_Functions contains the interfaces with the simulation
Forward_Model_Implementation is the main program which links these together.

Recommended that for now the simulations are not run and instead use the saved data that exists in this folder and is loaded. Vary the Inducing points, downsample, and other parameters to check accuracy of the model.
